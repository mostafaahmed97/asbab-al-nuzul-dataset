عَنِ الْبَرَاءِ رضي الله عنه قَالَ: ( لَمَّا نَزَلَتْ { لَّا يَسْتَوِى ٱلْقَٰعِدُونَ مِنَ ٱلْمُؤْمِنِينَ وَٱلْمُجَـٰهِدُونَ فِى سَبِيلِ ٱللَّهِ } قَالَ النَّبِيُّ ﷺ: « ادْعُ لِي زَيْداً، وَلْيَجِيءْ بِاللَّوْحِ وَالدَّوَاةِ وَالْكَتِفِ، - أَوْ، الْكَتِفِ وَالدَّوَاةِ - » ثُمَّ قَالَ: اكْتُبْ: { لَّا يَسْتَوِى ٱلْقَـٰعِدُونَ }. وَخَلْفَ ظَهْرِ النَّبِيِّ ﷺ عَمْرُو بْنُ أُمِّ مَكْتُومٍِ الْأَعْمَى، قَالَ: يَا رَسُولَ اللهِ، فَمَا تَأْمُرُنِي، فَإِنِّي رَجُلٌ ضَرِيرُ الْبَصَرِ؟ فَنَزَلَتْ مَكَانَهَا: {   لَّا يَسْتَوِى ٱلْقَـٰعِدُونَ مِنَ ٱلْمُؤْمِنِينَ غَيْرُ أُو۟لِى ٱلضَّرَرِ وَٱلْمُجَـٰهِدُونَ فِى سَبِيلِ ٱللَّهِ } ) واللفظ للبخاري.

وجاء عَنِ ابْنِ عَبَّاسٍ رضي الله عنهما شبيهاً به.
